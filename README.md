# Audiobook Generator

Welcome to the Audiobook Generator project! This project is designed to convert PDF files into audiobooks. It is built using Python and the PyPDF2 and pyttsx3 libraries. 

## Features
- Convert PDF files into audiobooks.
- The generated audiobook will be saved as mp3 in the current working directory.

## How to use
1. Replace the "samplepdf.pdf" with the desired pdf file name that you want to convert to audiobook
2. Run the script using python.

## Tech Stack
- Python
- PyPDF2
- pyttsx3
